# Pokedex_Explorer

## J10 – Pokedex Explorer

Mini projet JavaScript permettant d’explorer un Pokédex à partir d’un fichier JSON. 
Objectif : manipuler des données, travailler avec des tableaux et objets, et pratiquer les fonctions en JavaScript.

Objectifs pédagogiques Ce projet m’a permis de pratiquer : 
- La lecture de fichiers JSON avec fs.readFileSync() 
- La manipulation de tableaux et d’objets 
- L’utilisation de boucles et fonctions 
- Le tri et le filtrage de données 
- La recherche dynamique d’informations 

Les données proviennent du fichier JSON officiel : https://raw.githubusercontent.com/Biuni/PokemonGO-Pokedex/master/pokedex.json
